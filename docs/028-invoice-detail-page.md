# 028 - 請求書詳細ページ作成

## 概要

請求書の詳細表示画面を作成。
**骨組み重視**: 基本的な情報表示のみ、PDF出力やメール送信は後回し。

## 対象範囲

### 画面仕様
- URL: `/invoices/[id]`
- 請求書の基本情報を表示
- 明細リストを表示
- 金額計算表示

### 表示項目
**基本情報**
- 請求書番号
- 発行日
- 請求先名

**明細情報**
- 品目名
- 数量
- 単価  
- 金額

**合計金額**
- 合計金額（税込み計算は後で追加）
